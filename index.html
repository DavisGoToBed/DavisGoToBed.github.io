Over mij

Ik ben DavisGoToBed, een Java Developer die Minecraft servers helpt met unieke events, handige tools, RP-systemen en fun features. Ik maak plugins die makkelijk te configureren zijn, netjes werken op productie-servers en goed aanvoelen voor spelers én staff.

Mijn plugins

✨ DRG ✨

Red Light / Green Light event-plugin geïnspireerd op Squid Game — voor alle soorten servers (Roleplay tot Survival).

  • Event: Host start met /drg start en stopt met /drg stop; spelers binnen radius doen mee.
  • GREEN LIGHT: rennen richting de finish-lijn.
  • RED LIGHT: stil staan (0.5s grace). Beweeg je? Dan lig je eruit en ga je tijdelijk Spectator (max 20 blocks van host).
  • Finish = Host Line: finish-lijn links/rechts van host startpositie; raak lijn = SAFE + WIN.
  • Gear & Inventory: host oranje suit, deelnemers groen; lege inventory; na afloop/relog restore.
  • BossBar UI: live status + countdown voor GREEN/RED.
Spigot / Paper 1.21.x Java 21 Event

✨ DBackup ✨

Betrouwbare backup-plugin voor Minecraft servers — simpel, snel en volledig instelbaar.

  • Auto backups om de X minuten
  • Handmatige backup via /backup
  • Wereld saven + kopiëren
  • Configurable via config.yml
Spigot / Paper 1.21.x Java 21/17 Utility

✨ DFunDeaths ✨

Grappige death-message plugin met extra effects voor maximale chaos.

  • Custom Death Messages met woordgrappen
  • Death Effect: lightning + particles + sound (zichtbaar voor iedereen)
Spigot / Paper 1.21.x Java 21 Fun

✨ DShop ✨

Professioneel, veilig & uitbreidbaar shop- en sell-systeem voor Minecraft survival servers.

  • Shop GUI: /shop met categorieën, pagina’s, thema’s en volledige config via config.yml.
  • Bevestiging & hoeveelheid: confirm menu, stacks/amounts, quick buttons, live prijsberekening.
  • Sell systeem: /sell GUI, /sell hand, /sell all; tiers (IRON/DIAMOND/NETHERITE), blacklist, overrides, confirm.
  • Economy-safe: Vault support, inventory check, transaction-safe, geen exploits/duplication.
  • Exploit-proof GUI: shift-click/number keys/drag geblokkeerd waar nodig.
Spigot / Paper 1.21.x Java 21 Survival Economy

✨ DPolitieMenu ✨

Professioneel politie- & dossiersysteem voor RP-Minecraft servers.

  • Politie dashboard: /politie met overzicht van alle spelers (skull, wanted-status, laatste info).
  • Dossiers: last seen + locatie (~15 min geleden) met tijden in Europe/Amsterdam.
  • Strafblad: boetes & celstraffen met reden en geschiedenis (SQLite).
  • History GUI: record bekijken en verwijderen met bevestiging (permanent).
  • Command: /strafblad <speler> <boete|cel> <aantal> <reden>
Spigot / Paper 1.21.x Java 21 Roleplay

✨ DWasmachine ✨

De ultieme wasmachine-plugin voor je Minecraft server.

  • Custom block: kies welk block de wasmachine is via config.yml; rechtsklik om GUI te openen.
  • Custom GUI: input, wastablet en output met progress bar.
  • Wassen: damaged armor + Wastablets; output is 100% gerepareerd.
  • Veilig gedrag: GUI sluiten pauzeert, opnieuw openen gaat verder; geen item-loss.
  • Particles & sounds tijdens het wassen en bij completion.
  • Staff tools: /wastablet [player] [amount]
Spigot 1.21.x Java 21 Fun Utility

✨ DCoreDisplay ✨

Display-plugin met MOTD, server icon, animated tablist en staff GUI.

  • MOTD: meerdere regels, kleuren, emoji’s, placeholders.
  • Server icon: makkelijk aanpassen (64x64 PNG).
  • Animated tablist: themes, header/footer via config, live updates.
  • Staff GUI: beheren via /dcoredisplay of /dcd, reload met /dcoredisplay reload.
  • PlaceholderAPI support
Spigot 1.21.x Java 21 PlaceholderAPI
