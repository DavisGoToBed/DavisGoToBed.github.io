Over mij

Ik ben DavisGoToBed, een Java Developer die Minecraft servers helpt met unieke events, handige tools en fun features. Ik maak plugins die makkelijk te configureren zijn, netjes werken op productie-servers en goed aanvoelen voor spelers én staff.

Mijn plugins

✨ DRG ✨

Red Light / Green Light event-plugin geïnspireerd op Squid Game — voor alle soorten servers (RP tot Survival).

  • Squid Game Event: Host start met /drg start en stopt met /drg stop; alleen spelers in radius doen mee.
  • GREEN LIGHT: rennen richting de finish-lijn.
  • RED LIGHT: stil staan (0.5s grace). Beweeg je? Dan lig je eruit en ga je tijdelijk Spectator (max 20 blocks van host).
  • Finish = Host Line: finish-lijn links/rechts van de host startpositie; raak lijn = SAFE + WIN.
  • Event Gear & Inventory: host oranje suit, deelnemers groen; iedereen lege inventory; na afloop/relog krijg je alles terug.
  • BossBar UI: live status + countdown voor GREEN/RED.
Spigot / Paper 1.21.x Java 21 Event Plugin

✨ DBackup ✨

Betrouwbare backup-plugin voor Minecraft servers — simpel, snel en volledig instelbaar.

  • Automatische backups om de X minuten
  • Handmatige backup via /backup
  • Wereld saven + kopiëren (veilig en consistent)
  • Configurable via config.yml
Spigot / Paper 1.21.x Java 21/17 Utility

✨ DFunDeaths ✨

Grappige death-message plugin met extra effects voor maximale chaos (op een leuke manier).

  • Custom Death Messages: woordgrappen gebaseerd op hoe iemand sterft
  • Death Effect: lightning + particles + sound bij death (zichtbaar voor iedereen)
